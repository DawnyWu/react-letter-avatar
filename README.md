# DEMO

```js
git clone https://github.com/DawnyWu/react-letter-avatar.git
npm install
cd example
npm start
```
在浏览器中打开 http://localhost:3001 即可查看例子


# 使用

```js
<ReactLetterAvatar 
  name="Dawny Wu"
  size={50}
  radius={20}
/>
```
